# 魔力网络群聊
description: 具有魔法能力者(魔法少女、怪人等)都能使用魔力网络进行交流

ability:
  时效性: 具有魔法能力者可以通过意念用魔力网络进行群聊,这样的群聊以光速进行
  私密性: 群聊是私密的,不是群聊member的人无法感知到群聊在进行,也无法得知群聊内容,更无法在群聊中对话
  直白性: 魔法少女们在群聊中不会假装不知道<user>的过去
